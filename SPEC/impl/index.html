受信するRPN

0000:ピッチベンドセンシティビティのみ受信する。

プログラムバンク

バンクはMSBのみ受信する

バンク番号
MSB(10進数)		 バンク名 備考
0 Bank 0.7 htsfms0.7相当
96 Bank 0.9 htsfms0.9相当
一部正常に鳴らない音色あり
97 Bank 0.5 htsfms0.5相当
98 User Bank ユーザー音色
未設定の場合音が出ません

注意

本音源のFM音源はLFOやノイズ発振などの機能がないため0.9相当の音色の一部は正常に鳴らすことができません。
